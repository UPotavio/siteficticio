Já é possível viajar para Marte

Empresas americanas conseguem levar pessoas para outros planetas

Fonte: Empresa de viagens interplanetárias

Essa viagem está ao alcance de todos, com preços especiais!
A partir de 500 dólares, a viagem tem uma duração total de 7 horas.

Mas muita atenção: são necessários preparativos antes de poder viajar.

O que é necessário para poder viajar?

Para viajar, é necessário que:

  • Exames médicos
  • Duas semanas de treino especial
  • Passaporte interplanetário válido
  • Assinar um termo de regulamento

Procedimentos necessários para viajar

  1. Escolher a data da viagem
  2. Efetuar 50% do pagamento antes da viagem
  3. Fazer o treino especial
  4. Ter feito os exames médicos
  5. Assinar o termo de regulamento

MUITO IMPORTANTE

A viagem é apenas para maiores de idade dos seus países.
